# text-analyzer
Une un simple éditeur de texte avec détection de langue

## Installation
Clonez ce depôt 
```bash
$ git clone https://github.com/Capacity-Dev/text-analyzer

$ cd text-analyzer
```
installez les dependance et lancez l'application

```bash
$ yarn
$ yarn dev
```
ou
```bash
$ npm install
$ npm run dev
````
